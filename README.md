# PyManager - Python 脚本管理器

专为 **Python 新手** 打造的脚本管理工具，让脚本管理变得简单直观！

## ✨ 三大核心亮点

### 1. 🎯 新手友好的脚本管理

PyManager 提供直观的图形界面，即使您是 Python 初学者也能轻松上手：

- **简单操作**：双击运行脚本，无需记住复杂命令
- **智能分组**：自动按文件夹组织脚本，一目了然
- **一键收藏**：重要脚本收藏置顶，快速访问
- **批量管理**：支持多选脚本进行批量操作

### 2. 🧩 自动依赖检测与安装

从网上下载的脚本，无需手动安装依赖：

- **智能识别**：自动分析脚本中的 `import` 语句
- **一键安装**：运行前自动检测并安装所需 Python 包
- **版本兼容**：自动处理包冲突，选择合适版本
- **进度显示**：实时显示依赖安装进度，心中有数

### 3. ⭐ GitHub 高星项目推荐

发现优质 Python 资源从未如此简单：

- **高星项目推荐**：浏览 GitHub 上最受欢迎的 Python 项目
- **智能搜索**：按关键词、星标数、更新时间筛选
- **一键下载**：直接下载项目到本地使用
- **AI 分析**：智能分析项目功能和使用场景

## 🚀 快速开始

### 系统要求

- **操作系统**: Windows 7/10/11
- **Python 版本**: Python 3.7 或更高版本

### 安装步骤

1. **下载项目**
   ```bash
   git clone https://github.com/zhangyapu1/pymanager.git
   cd pymanager
   ```
2. **安装依赖包**
   ```bash
   pip install -r requirements.txt
   ```
3. **启动应用程序**
   ```bash
   python main.pyw
   ```
   或者直接双击 `main.pyw` 文件启动

## 🎮 使用教程

### 添加脚本

1. 点击「添加脚本」按钮，或直接拖拽 `.py` 文件到窗口
2. 脚本会自动归类到对应分组

### 运行脚本

1. 双击列表中的脚本即可运行
2. 运行前会自动检查并安装依赖
3. 输出信息显示在底部窗口

### 探索 GitHub 脚本市场

1. 点击「脚本市场」按钮
2. 在搜索框输入关键词，或浏览高星项目
3. 点击项目查看详情，选择文件下载
